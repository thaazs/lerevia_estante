Design Thinking: uma metodologia poderosa para decretar o fim das velhas ideias

Tim Brown

Design Avançado

Este livro é um guia para líderes criativos que desejam implementar o design thinking em suas organizações. Mais do que um manual para designers, ele propõe uma abordagem inovadora para solucionar problemas, explorando novas alternativas para os negócios e a sociedade. Escrito com imaginação, humor e visão, esta obra inspiradora é essencial para quem busca enfrentar os desafios atuais e transformar essas dificuldades em oportunidades para o futuro. É uma leitura fundamental para qualquer profissional interessado em impulsionar a inovação e a criatividade.
